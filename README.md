# C++ 迷宫寻路系统

一个功能完整的C++迷宫寻路系统，实现了多种路径寻找算法并提供图形化可视化界面。

## 🎯 项目特性

### 核心功能
- **迷宫生成**: 支持随机生成和DFS算法生成迷宫
- **路径寻找**: 实现了DFS、BFS和A*三种寻路算法
- **性能比较**: 提供算法性能对比分析
- **可视化展示**: ASCII字符界面和HTML网页可视化
- **动画演示**: 路径寻找过程的动画展示
- **结果导出**: 支持文本和HTML格式导出

### 技术特点
- **数据结构**: 使用二维矩阵表示迷宫，栈和队列实现搜索算法
- **算法优化**: A*算法使用曼哈顿距离启发式函数
- **内存管理**: 智能指针管理内存，防止内存泄漏
- **模块化设计**: 清晰的类结构，易于扩展和维护

## 🎮 使用说明

### 主要功能

1. **创建迷宫**
   - 随机生成迷宫
   - DFS算法生成迷宫
   - 预设示例迷宫

2. **求解迷宫**
   - 深度优先搜索 (DFS)
   - 广度优先搜索 (BFS) - 保证最短路径
   - A*算法 - 启发式搜索
   - 查找所有可能路径

3. **算法比较**
   - 同时运行三种算法
   - 比较路径长度、搜索时间、访问节点数
   - 显示最优算法

4. **可视化展示**
   - 控制台彩色显示
   - HTML网页可视化
   - 动画演示搜索过程

5. **结果导出**
   - 导出迷宫到文本文件
   - 生成HTML可视化文件
   - 保存路径信息

### 符号说明
- `██` - 墙壁
- `  ` - 可通行路径
- `SS` - 起点/入口
- `EE` - 终点/出口
- `**` - 最终路径
- `··` - 搜索过的路径

## 🧮 算法说明

### 1. 深度优先搜索 (DFS)
- **原理**: 沿着一条路径尽可能深入，遇到死路则回溯
- **特点**: 不保证最短路径，但内存消耗小
- **时间复杂度**: O(V + E)
- **空间复杂度**: O(V)

### 2. 广度优先搜索 (BFS)
- **原理**: 逐层搜索，先访问距离起点近的节点
- **特点**: 保证找到最短路径
- **时间复杂度**: O(V + E)
- **空间复杂度**: O(V)

### 3. A*算法
- **原理**: 使用启发式函数引导搜索方向
- **启发式函数**: 曼哈顿距离
- **特点**: 在保证最优解的前提下提高搜索效率
- **时间复杂度**: O(b^d) (b为分支因子，d为深度)
